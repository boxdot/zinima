# zinima

A port of [Jekyll]'s [minima] theme to [Zola] with focus on being *accurate*.

![screenshot](screenshot.png)

## Differences

* no Google analytics
* no SEO tag
* code blocks are styled and highlighted natively by Zola
* support for [last.fm](https://last.fm) in social links
